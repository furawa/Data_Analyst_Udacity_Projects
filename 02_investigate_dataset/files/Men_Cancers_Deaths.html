Summary¶

There are 5 common types of cancer in Men, lung cancer, stomach cancer, liver cancer, prostate cancer and colon&rectum cancer. In 2016, more than 1 million Men die of one of this cancer over the world. Which country is affected the most by these cancers? What is the cancer which kill the most? How is the evolution of these cancer deaths since 1990?
In this project we will analyse the data taken from the Gapminder World about these 5 types of cancer in Men and try to find an answer to these questions.

# Transform the files from wide to long with a function

def gather(df:"DataFrame", value_name:str) -> "df": 
    """function to transform and rename the columns """
    # This reshape the data from wide to long, renaming the columns 
    df = df.melt(id_vars = ["country"],var_name = "year", value_name = value_name + "deaths") 
    return df   

# Use the function gather to create "long" dataframes

list_df = []    # Create an empty list to contain the new "long" dataframes

# Use a loop to access all the files in the dictionnary using gather function 
for name,df in all_dict.items():                     
    name = gather(df,name[:-2]) # name[:-2] -> removes "df" sting from each name
    list_df.append(name)        # Put all the new dataframe in the list 
    
# Use the reduce function to merge all the df in a row    
df = reduce(lambda left,right: pd.merge(left,right, on = ["country", "year"]), list_df) 

# Write to a csv for future uses
df.to_csv("files/common_men_cancers_deaths.csv")

Conclusions¶

Despite of the evolution of the technology and medecin, every year million Men die of the most common cancer, the number of deaths increases each year. The lung cancer in overall is the most dangerous and the prostate cancer is the one killing the less. But further analysis are needed because even if the number of deaths is increasing each year the population are increasing too and this must be taken under consideration, it will be appropriate to work with the proportion (number_of_deaths_per_year/population_per_year) to have a better result.
What could be the causes of the increase of all those cancers will be the subject of another analysis.
